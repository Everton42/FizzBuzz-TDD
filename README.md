# FizzBuzz-TDD
Projeto para aprender sobre TDD com XUnit

O Fizz Buzz é um jogo de palavras em grupo para as crianças ensinarem sobre a divisão.
Os jogadores se revezam para contar incrementalmente,
substituindo qualquer número divisível por três pela palavra "fizz"
e qualquer número divisível por cinco com a palavra "buzz".

## Regras do Projeto

  Dado um número nmbr, imprima a contagem até o mesmo. 
   - Se algum número da sequência for divisível por 3,
  substitua por "Fizz". 
   - Se for divisível por 5, substitua por "Buzz. 
   - Se for divisível por 3 e 5, substitua por "FizzBuzz.
  
  Ex:
    nmbr = 15 >> 1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz
